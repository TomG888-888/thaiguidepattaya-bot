# Правила работы с фото

1. Использовать фото из существующих товаров ВК, если права и источник понятны.
2. Использовать фото поставщика, когда поставщик разрешил размещение в карточках, постах и сообщениях.
3. Использовать free-license фото только при понятной лицензии и возможности указать источник.
4. AI-фото генерировать только с нуля по описанию, без копирования чужих отельных, туристических или коммерческих кадров.
5. Не использовать чужие фото с отелей/сайтов без разрешения, включая агрегаторы, соцсети и рекламные страницы.
6. Не хранить net/internal данные в публичных фото-описаниях, подписях, названиях файлов, alt-текстах и источниках.

## Имена файлов в карточках туров

В каждой папке тура внутри `static/tours/` использовать ровно такие имена:

- `cover.jpg`
- `gallery_1.jpg`
- `gallery_2.jpg`
- `gallery_3.jpg`
- `gallery_4.jpg`
